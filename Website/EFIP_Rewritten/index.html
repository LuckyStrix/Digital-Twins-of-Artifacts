Digital Twins of Artifacts

Starting as Freshman Imaging Project and continued as Extended-FIP, we have created a comprehensive system for creating 3D models/replicas of historical artifacts. This website contains the viewers to see the artifacts we have imaged, as well as links to all of our documentation.

Artifacts

Every digitized artifact, grouped by type. Drop a new folder into artifacts/ and rerun tools/build_manifest.py to add it here — no page edits needed.
